OpenGL环境搭建

发表于 2020-09-26 更新于 2021-09-09

进行Visual Studio的OpenGL开发环境搭建

下载必要库

  • GLFW
    • 针对OpenGL的C语言库
    • 下载: Doenload | GLFW
    • 下载32位版本, 因为64位版本不稳定
  • GLEW
    • 用于代替GLAD来管理函数指针
    • 下载: The OpenGL Extension Wrangler Library download | SourceForge.net

将项目与库连接

  1. 将GLFW和GLEW解压到一个新文件夹中，文件夹名建议为OpenGL, 目录结构如下

  2. 使用Visual Studio创建项目, 创建一个.cpp格式的文件, 比如main.cpp

  3. 设置项目属性

    1. 配置: 所有配置
    2. 平台: 所有平台
    3. C/C++ -> 常规 -> 附加包含内容 -> 编辑 -> 新行
      1. \OpenGL\glfw-x.x.x.bin.WIN32\include
      2. \OpenGL\glew-x.x.x\include
    4. 链接器 -> 常规 -> 附加库目录 -> 编辑 -> 新行
      1. \OpenGL\glfw-x.x.x.bin.WIN32\lib-vc20xx
      2. \OpenGL\glew-x.x.x\lib\Release\Win32
    5. 链接器 -> 输入 -> 附加依赖项 -> 编辑
      1. opengl32.lib
      2. glfw3.lib
      3. glew32s.lib
